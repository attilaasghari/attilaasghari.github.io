The Sorting Algorithms Simulator is an interactive Python application that provides real-time visualization of various sorting algorithms. Built with Tkinter for the GUI and Matplotlib for data visualization, this project offers an engaging way to understand how different sorting algorithms work.

**Key Features:**
- Visualize 6+ sorting algorithms (Bubble Sort, Insertion Sort, Selection Sort, Merge Sort, Quick Sort, Heap Sort)
- Real-time progress tracking with performance metrics
- Customizable array size (3-500 elements)
- Interactive controls: Start, Stop, Reset functionality
- Detailed value labels and array display
- About section with developer information

**Technical Highlights:**
- Object-oriented Python implementation
- Generator-based animation system for smooth visualization
- Dynamic array generation and reset capability
- Comprehensive error handling and input validation
- Clean, modular code structure

**Educational Value:**
This project serves as an excellent resource for:
- Computer science students learning algorithms
- Educators demonstrating sorting concepts
- Developers interested in Python GUI programming
- Anyone curious about algorithm visualization
